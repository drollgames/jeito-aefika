# SdK Api

### Start discord afk:

JS:

```js
var ga = require("./v3/http/discord"); // uses gslk-js
ga.run();
```

* Resposta

`.afk()` ou `.gaAfk()`

### query

* GsLK: https://api.gslk.ga/v1/get-afk/{Action}
* Self: https://get-afk.tk/query/v1/{Action
* Proxy (GsLK): https://gslk.ga/proxy/na-1/get-afk/{Action]
* API: *[Legacy]*
